准备工作

  • 安装nodejs
  • 安装git
  • 注册github（处理好ssh key）

开始hexo

  • 安装hexo （目测要翻墙）
    • npm install -g hexo
  • 初始化hexo项目
    • 找个合适的目录 hexo init
    • 会下载项目需要的文件 需要些事件 等着呗(‧‧)nnn

配置hexo

  • 修改配置文件 _config.xml 配置随心情改啦
    • 修改全局配置时，注意缩进，同时注意冒号后面要有一个空格

  • 关联github

  • 安装git-deployer插件

    • npm install hexo-deployer-git --save

常用命令

  • hexo clean
  • hexo generate / hexo g 生成页面
  • hexo deploy 发布到远程仓库
  • hexo new "文件名" 新建文章
  • hexo new page '页面'
  • hexo clean
  • hexo clean
